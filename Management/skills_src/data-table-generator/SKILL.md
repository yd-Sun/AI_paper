# 数据表格生成器

> 根据研究数据生成规范的学术表格，支持描述性统计、对比分析等多种表格类型。

## 功能概述

学术论文中的表格需要遵循特定的规范。本技能帮助你：
- 将原始数据转换为规范的学术表格
- 自动生成表格标题和注释
- 支持多种表格类型（描述性统计、对比分析、相关性矩阵等）
- 确保表格符合学术规范

适合在数据分析完成后使用，快速生成符合期刊要求的表格。

## 主要功能

### 功能1：生成表格

**用途**：根据数据和要求生成格式化的学术表格。

**使用场景**：
- 有原始数据，需要制作表格
- 想要快速生成规范的表格
- 不确定表格格式是否正确

**使用示例**：

**示例1：描述性统计表**
```
原始数据：
实验组：n=60, 平均分=75.3, 标准差=8.2, 最小值=58, 最大值=92
对照组：n=60, 平均分=68.5, 标准差=9.1, 最小值=52, 最大值=88

表格类型：描述性统计表

表格标题：两组学生批判性思维测试得分的描述性统计

添加注释：是

生成结果：

表1 两组学生批判性思维测试得分的描述性统计

| 组别 | 样本量 | 平均分 | 标准差 | 最小值 | 最大值 |
|------|--------|--------|--------|--------|--------|
| 实验组 | 60 | 75.3 | 8.2 | 58 | 92 |
| 对照组 | 60 | 68.5 | 9.1 | 52 | 88 |

注：实验组为使用AI工具的学生，对照组为未使用AI工具的学生。满分为100分。

【格式说明】
✓ 表格标题在表格上方
✓ 使用三线表格式
✓ 数据保留一位小数
✓ 注释说明了关键信息
```

**示例2：对比分析表**
```
原始数据：
前测：实验组M=65.2, SD=7.8; 对照组M=64.8, SD=8.1
后测：实验组M=75.3, SD=8.2; 对照组M=68.5, SD=9.1

表格类型：对比分析表

表格标题：两组学生前后测批判性思维得分对比

生成结果：

表2 两组学生前后测批判性思维得分对比

| 组别 | 前测 M(SD) | 后测 M(SD) | 提升幅度 | t值 | p值 |
|------|------------|------------|----------|-----|-----|
| 实验组 | 65.2(7.8) | 75.3(8.2) | +10.1 | 8.32 | <.001 |
| 对照组 | 64.8(8.1) | 68.5(9.1) | +3.7 | 2.45 | .016 |

注：M = 平均分，SD = 标准差。***p < .001, **p < .01, *p < .05。

【格式说明】
✓ 均值和标准差合并显示
✓ 提升幅度用+/-表示
✓ p值用<表示显著性
✓ 注释说明了统计符号
```

**示例3：相关性矩阵**
```
原始数据：
变量1与变量2：r=0.65, p<0.001
变量1与变量3：r=0.42, p=0.003
变量2与变量3：r=0.38, p=0.012

表格类型：相关性矩阵

表格标题：各变量之间的相关系数矩阵

生成结果：

表3 各变量之间的相关系数矩阵

| 变量 | 1 | 2 | 3 |
|------|---|---|---|
| 1. 批判性思维 | - | | |
| 2. 写作质量 | .65*** | - | |
| 3. 使用频率 | .42** | .38* | - |

注：***p < .001, **p < .01, *p < .05。

【格式说明】
✓ 对角线为"-"
✓ 只显示下三角
✓ 相关系数保留两位小数
✓ 显著性用星号标注
```

```
不同意：12人（10.0%）

表格类型：频数分布表

表格标题：学生对AI工具使用态度的频数分布

生成结果：

表4 学生对AI工具使用态度的频数分布（N=120）

| 态度 | 频数 | 百分比 | 累积百分比 |
|------|------|--------|------------|
| 非常同意 | 25 | 20.8% | 20.8% |

【格式说明】

✓ 百分比保留一位小数
```

**参数说明**：
- **原始数据**（必填）：粘贴原始数据，支持多种格式
- **表格类型**（必填）：选择表格类型
- **表格标题**（可选）：如果不提供，系统会自动生成
- **添加注释**（可选）：是否在表格下方添加注释

**注意事项**：
- 数据要准确，生成后务必检查
- 不同期刊对表格格式的要求可能不同
- 表格标题应该简洁明了，能够独立理解
- 注释要说明关键信息（如样本量、显著性水平等）

## 使用技巧

1. **三线表**：学术论文通常使用三线表（只有顶线、底线和表头线）
2. **标题位置**：表格标题在表格上方，图片标题在图片下方
3. **数据精度**：根据数据性质决定小数位数，通常1-2位
4. **对齐方式**：文字左对齐，数字右对齐或居中
5. **注释规范**：用注释说明缩写、符号、显著性水平等

## 常见问题

**Q: 表格和图片如何选择？**

A: 一般原则：
- 数据量少（<10个数据点）：用文字描述
- 需要精确数值：用表格
- 展示趋势或关系：用图片
- 复杂数据：表格+图片

**Q: 表格标题如何写？**

A: 好的表格标题应该：
- 简洁明了（通常一句话）
- 包含关键信息（变量、样本等）
- 能够独立理解（不看正文也能懂）
- 示例："不同教学方法对学习效果的影响（N=120）"

**Q: 如何报告统计显著性？**

A: 常用方法：
- 星号标注：*p<.05, **p<.01, ***p<.001
- 直接报告p值：p=.023
- 在注释中说明显著性水平

**Q: 表格太大怎么办？**

A: 解决方法：
- 简化：只保留关键数据
- 拆分：分成多个小表格
- 附录：详细数据放附录
- 在线补充材料：提供完整数据

**Q: 如何处理缺失数据？**

A: 
- 用"-"或"N/A"表示
- 在注释中说明缺失原因
- 报告缺失数据的比例

## 学术表格规范

### 基本要求
1. **三线表**：只有顶线、底线和表头线
2. **标题在上**：表格标题在表格上方
3. **编号连续**：表1、表2、表3...
4. **独立理解**：不看正文也能理解表格内容

### 数据呈现
1. **对齐**：文字左对齐，数字右对齐或居中
2. **精度**：根据数据性质决定小数位数
3. **单位**：在表头或标题中说明单位
4. **符号**：统一使用标准符号（如M、SD、r等）

### 注释规范
1. **位置**：在表格下方
2. **内容**：说明缩写、符号、显著性水平、特殊情况等
3. **格式**：注：xxxxx

### 常见错误
❌ 表格线太多（不要用网格线）
❌ 标题太长或太短
❌ 数据精度不一致
❌ 缺少必要的注释
❌ 表格太大，难以阅读
