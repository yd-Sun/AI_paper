# 摘要聚焦增强器

> 用于约束摘要的结构完整性，突出研究目的、方法、结果和结论。

## 功能概述

摘要是论文的"门面"，很多读者（包括审稿人）会先看摘要来决定是否继续阅读全文。一个好的摘要应该：
- 结构完整，包含目的、方法、结果、结论四要素
- 简洁明了，通常在200-300字（中文）或150-250词（英文）
- 独立完整，不看正文也能理解研究的核心内容

本技能帮助你检查摘要是否符合学术规范，确保四要素齐全、表述清晰。

## 主要功能

### 功能1：审校摘要

**用途**：检查摘要是否包含目的、方法、结果和结论四个必要部分，并给出改进建议。

**使用场景**：
- 刚写完摘要，需要检查是否符合规范
- 期刊或导师要求修改摘要
- 不确定摘要是否完整

**使用示例**：
```
摘要正文：
本研究探讨了生成式AI工具对大学生学术写作的影响。通过对100名大学生的问卷调查，发现AI工具能够提高写作效率，但可能影响批判性思维能力。

学科方向：教育技术

审校结果：

结构分析：
✅ 目的：已包含（"探讨了生成式AI工具对大学生学术写作的影响"）
✅ 方法：已包含（"对100名大学生的问卷调查"）
⚠️ 结果：不够具体（"提高写作效率"、"影响批判性思维能力"需要量化）
❌ 结论：缺失（没有说明研究的意义或建议）

改进建议：

1. 目的部分：可以更具体
   原文："探讨了生成式AI工具对大学生学术写作的影响"
   建议："探讨生成式AI工具（如ChatGPT）对大学生学术写作效率和批判性思维能力的影响"

2. 方法部分：需要补充研究设计
   原文："对100名大学生的问卷调查"
   建议："采用准实验设计，对100名大学生进行为期8周的对照实验，通过问卷和写作样本分析"

3. 结果部分：需要量化数据
   原文："能够提高写作效率，但可能影响批判性思维能力"
   建议："实验组写作时间平均减少35%，但批判性思维测试得分比对照组低12%（p<0.05）"

4. 结论部分：需要补充
   建议补充："研究表明，AI工具在提高效率的同时，需要配合批判性思维训练，以避免对学生思维能力的负面影响。"

优化后的摘要：
本研究探讨生成式AI工具（如ChatGPT）对大学生学术写作效率和批判性思维能力的影响。采用准实验设计，对100名大学生进行为期8周的对照实验，通过问卷和写作样本分析数据。结果显示，实验组写作时间平均减少35%，但批判性思维测试得分比对照组低12%（p<0.05）。研究表明，AI工具在提高效率的同时，需要配合批判性思维训练，以避免对学生思维能力的负面影响。本研究为AI在教育中的合理应用提供了实证依据。
```

**参数说明**：
- **摘要正文**（必填）：粘贴你当前的摘要
- **学科方向**（可选）：不同学科对摘要的要求可能略有差异

**注意事项**：
- 摘要应该是**独立完整**的，不看正文也能理解
- 避免使用"本文"、"本研究"等指代词过多
- 不要在摘要中引用文献
- 数据要准确，不要夸大或模糊

## 使用技巧

1. **先写正文，再写摘要**：摘要是对全文的总结，应该在正文完成后再写
2. **使用结构化摘要**：有些期刊要求结构化摘要（明确标注目的、方法、结果、结论），即使不要求，也可以按这个结构写
3. **突出创新点**：在结论部分，要突出你的研究有什么新发现或新贡献
4. **控制字数**：中文摘要通常200-300字，英文摘要150-250词，不要超出期刊要求
5. **多次修改**：摘要是论文的门面，值得花时间反复打磨

## 常见问题

**Q: 摘要和引言有什么区别？**

A: 摘要是对整篇论文的浓缩，包括目的、方法、结果、结论；引言主要介绍研究背景、研究问题、研究意义，不包含结果和结论。

**Q: 摘要需要包含关键词吗？**

A: 关键词通常单独列出，不包含在摘要正文中。但摘要中应该自然出现一些关键词。

**Q: 摘要可以分段吗？**

A: 通常不分段，但如果期刊要求结构化摘要，可以按"目的、方法、结果、结论"分段或用小标题。

**Q: 摘要中可以使用缩写吗？**

A: 可以，但第一次出现时要写全称，如"人工智能（AI）"。常见的缩写（如AI、GDP）可以直接使用。

**Q: 摘要需要包含数据吗？**

A: 实证研究的摘要应该包含关键数据，如样本量、主要统计结果等。理论研究或综述论文可以不包含具体数据。

## 摘要四要素详解

### 1. 目的（Purpose/Objective）
- 说明研究的背景和目标
- 回答"为什么做这个研究"
- 通常1-2句话

示例：
"本研究旨在探讨生成式AI工具对大学生学术写作能力的影响。"

### 2. 方法（Methods）
- 说明研究设计、样本、工具、程序
- 回答"怎么做的研究"
- 通常2-3句话

示例：
"采用准实验设计，对100名大学生进行为期8周的对照实验，通过问卷和写作样本分析数据。"

### 3. 结果（Results）
- 呈现主要发现和数据
- 回答"发现了什么"
- 通常2-3句话，要有具体数据

示例：
"结果显示，实验组写作时间平均减少35%，但批判性思维测试得分比对照组低12%（p<0.05）。"

### 4. 结论（Conclusion）
- 总结研究的意义和贡献
- 回答"这个研究有什么用"
- 通常1-2句话

示例：
"研究表明，AI工具在提高效率的同时，需要配合批判性思维训练。本研究为AI在教育中的合理应用提供了实证依据。"
