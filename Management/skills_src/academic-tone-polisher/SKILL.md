# 学术语气校正器

> 用于章节写作和学术润色，约束表述风格、逻辑衔接和术语一致性。

## 功能概述

学术写作有其特定的语言风格和规范，与日常写作有很大不同。本技能帮助你：
- 识别并修正口语化表达
- 改善段落之间的逻辑衔接
- 确保术语使用的一致性
- 提升学术表达的专业性

适合在完成初稿后使用，通过系统的语气检查，让你的论文更加规范、专业。

## 主要功能

### 功能1：检查语气

**用途**：检查段落是否存在口语化、跳跃式表述或术语不一致问题，并提供修改建议。

**使用场景**：
- 初稿完成，需要润色学术表达
- 导师或审稿人指出语言问题
- 想要提升论文的专业性

**使用示例**：

**示例1：学术语气检查**
```
正文内容：
AI现在特别火，很多学生都在用。但是用AI写论文到底好不好呢？有人说好，有人说不好。我觉得这个问题挺复杂的，需要好好研究一下。

检查重点：学术语气

检查结果：

❌ 口语化问题：
1. "特别火" → 建议改为"受到广泛关注"或"应用日益普及"
2. "很多学生都在用" → 建议改为"大量学生使用"或"学生使用率显著提升"
3. "到底好不好呢" → 建议改为"其影响如何"或"是否有益"
4. "有人说好，有人说不好" → 建议改为"学术界对此存在不同观点"
5. "我觉得" → 建议改为"本研究认为"或"研究表明"
6. "挺复杂的" → 建议改为"较为复杂"或"涉及多个维度"
7. "好好研究一下" → 建议改为"进行深入研究"或"系统探讨"

修改后：
生成式AI工具在教育领域受到广泛关注，大量学生开始使用这类工具辅助学术写作。然而，学术界对其影响存在不同观点。鉴于该问题涉及多个维度，本研究拟对此进行系统探讨。
```

**示例2：逻辑衔接检查**
```
正文内容：
AI工具可以提高写作效率。学生的批判性思维能力很重要。教育者需要关注AI的使用。

检查重点：逻辑衔接

检查结果：

❌ 逻辑跳跃问题：
- 三句话之间缺少逻辑连接词
- 句子之间的关系不明确（并列？因果？转折？）
- 缺少过渡和铺垫

修改建议：
1. 明确句子之间的逻辑关系
2. 使用适当的连接词（然而、因此、此外等）
3. 增加过渡句

修改后：
AI工具可以显著提高写作效率，这为学生节省了大量时间。然而，过度依赖AI可能削弱学生的批判性思维能力，而批判性思维恰恰是学术写作的核心能力。因此，教育者需要在鼓励使用AI工具的同时，关注学生思维能力的培养。
```

**示例3：术语一致性检查**
```
正文内容：
本研究使用生成式AI工具进行实验。实验中，学生使用ChatGPT完成写作任务。结果显示，人工智能辅助写作能够提高效率。

检查重点：术语一致性

检查结果：

⚠️ 术语不一致：
- "生成式AI工具"、"ChatGPT"、"人工智能辅助写作" 指代同一事物，但表述不一致
- 建议统一使用一个术语，或在首次出现时说明

修改建议：
1. 首次出现时给出完整定义
2. 后续使用统一的简称
3. 如果需要区分，要明确说明

修改后：
本研究使用生成式AI工具（以ChatGPT为代表）进行实验。实验中，学生使用ChatGPT完成写作任务。结果显示，生成式AI工具能够显著提高写作效率。
```

**参数说明**：
- **正文内容**（必填）：粘贴需要检查的段落
- **检查重点**（必填）：选择学术语气、逻辑衔接或术语一致性

**注意事项**：
- 建议分段检查，每次不超过500字
- 不同学科的学术语气可能略有差异
- 修改时要保持原意，不要过度修改

## 使用技巧

1. **先整体后局部**：先检查整体逻辑，再检查具体表达
2. **多读几遍**：修改后大声朗读，看是否流畅
3. **参考范文**：找几篇本领域的优秀论文，学习其表达方式
4. **避免过度修饰**：学术写作追求准确、简洁，不要过度使用华丽辞藻
5. **保持一致性**：全文的语气风格应该保持一致

## 常见问题

**Q: 学术写作可以用"我"吗？**

A: 可以，但要适度。现代学术写作允许使用第一人称，但不要过度使用。建议用"本研究"、"研究者"等替代。

**Q: 学术写作是不是越复杂越好？**

A: 不是。学术写作追求的是**准确、简洁、清晰**，而不是复杂。能用简单词就不用复杂词，能用短句就不用长句。

**Q: 如何避免口语化？**

A: 常见的口语化表达：很、非常、特别、挺、有点、好像、可能、大概等。建议用更准确的词替代，如"显著"、"明显"、"在一定程度上"等。

**Q: 逻辑连接词有哪些？**

A: 常用的逻辑连接词：
- 并列：此外、同时、另外
- 递进：进一步、更重要的是
- 转折：然而、但是、尽管如此
- 因果：因此、由此可见、鉴于
- 总结：综上所述、总之

**Q: 术语首次出现时需要解释吗？**

A: 是的。专业术语或缩写首次出现时，应该给出完整定义或解释，如"生成式人工智能（Generative AI，以下简称生成式AI）"。

## 学术写作常见问题

### 口语化表达对照表

| 口语化 | 学术化 |
|--------|--------|
| 很多 | 大量、众多、相当数量 |
| 很重要 | 至关重要、具有重要意义 |
| 好像 | 似乎、可能、倾向于 |
| 差不多 | 大致、约、近似 |
| 越来越 | 日益、逐渐增加 |
| 特别 | 尤其、特别是、显著 |
| 挺好的 | 较为理想、效果良好 |

### 逻辑连接词使用指南

**表示并列关系**：
- 同时、此外、另外、与此同时、同样

**表示递进关系**：
- 进一步、更重要的是、不仅...而且、甚至

**表示转折关系**：
- 然而、但是、尽管如此、相反、与此相反

**表示因果关系**：
- 因此、由此可见、鉴于、基于、导致

**表示总结关系**：
- 综上所述、总之、总体而言、概括来说
