# 研究方法设计器

> 根据研究问题设计完整的研究方法论蓝图，包括研究范式、方法选择、数据策略、分析框架和效度标准。

## 功能概述

研究方法是连接研究问题和研究结论的桥梁。一个科学的研究方法设计应该：
- 与研究问题相匹配
- 具有可操作性
- 能够保证研究的信度和效度
- 符合学科规范和伦理要求

本技能帮助你从零开始设计完整的研究方法，或评估现有方法设计的合理性。

## 主要功能

### 功能1：设计研究方法

**用途**：根据研究问题，设计完整的研究方法论方案，包括研究范式、设计类型、样本选择、数据收集、分析方法等。

**使用场景**：
- 确定了研究问题，但不知道用什么方法研究
- 需要系统的方法论指导
- 想要确保方法设计的科学性

**使用示例**：
```
研究问题：
使用生成式AI工具辅助学术写作，对大学生的批判性思维能力有何影响？

学科领域：教育学

研究类型偏好：混合研究

研究限制：
- 时间：6个月
- 样本：可以在本校招募100-150名学生
- 资源：有问卷调查和写作样本分析的条件

输出示例：

【研究方法设计方案】

一、研究范式与设计
- 研究范式：实用主义范式（混合研究）
- 研究设计：准实验设计（前后测对照组设计）
- 研究周期：8周干预 + 4周追踪

二、研究对象与样本
- 总体：某大学本科生
- 样本量：120人（实验组60人，对照组60人）
- 抽样方法：方便抽样 + 随机分组
- 纳入标准：大二或大三学生，有学术写作经验
- 排除标准：已经频繁使用AI工具的学生

三、研究变量
- 自变量：AI工具使用（使用 vs 不使用）
- 因变量：批判性思维能力（论证质量、逻辑连贯性）
- 控制变量：年级、专业、写作基础

四、数据收集
定量数据：
1. 批判性思维测试（前测、后测）
2. 写作质量评分（使用评分量表）
3. 使用行为数据（使用频率、时长）

定性数据：
1. 学生访谈（10-15人）
2. 写作过程观察记录
3. 反思日志

五、数据分析方法
定量分析：
- 描述性统计（均值、标准差）
- 独立样本t检验（组间差异）
- 配对样本t检验（前后测差异）
- 协方差分析（控制变量影响）

定性分析：
- 主题分析法
- 内容分析法

六、信度与效度保证
- 内部效度：随机分组、控制变量
- 外部效度：样本代表性、结果可推广性
- 信度：测量工具的一致性、评分者信度
- 效度：内容效度、构念效度

七、伦理考虑
- 知情同意：所有参与者签署知情同意书
- 隐私保护：数据匿名化处理
- 自愿原则：可随时退出研究
- 伦理审查：提交学校伦理委员会审查

八、研究局限性
- 样本局限：仅限于一所大学
- 时间局限：8周干预可能不够长
- 测量局限：批判性思维难以完全量化
```

**参数说明**：
- **研究问题**（必填）：你的核心研究问题
- **学科领域**（必填）：不同学科的方法论传统不同
- **研究类型偏好**（可选）：定量、定性或混合研究
- **研究限制**（可选）：时间、资源、样本等限制条件

**注意事项**：
- 方法要与问题匹配，不要为了用某种方法而用
- 考虑实际可行性，不要设计过于理想化的方案
- 注意伦理问题，特别是涉及人类受试者的研究

### 功能2：评估研究方法

**用途**：评估现有研究方法设计的合理性和可行性，指出潜在问题并提供改进建议。

**使用场景**：
- 已经设计了研究方法，需要专业评估
- 导师或审稿人对方法提出质疑
- 想要确保方法的科学性

**使用示例**：
```
研究方法描述：
我打算用问卷调查的方式，调查学生对AI工具的看法，然后分析AI对学生的影响。

研究问题：
AI工具对学生批判性思维能力有何影响？

评估结果：

❌ 方法与问题不匹配
- 问题：研究问题关注的是"影响"（因果关系），但问卷调查只能了解"看法"（态度），无法建立因果关系
- 建议：应该使用实验设计或准实验设计，对比使用和不使用AI的学生

⚠️ 测量方法不明确
- 问题："批判性思维能力"如何测量？用什么工具？
- 建议：使用标准化的批判性思维测试，或设计写作任务并用评分量表评估

❌ 缺少对照组
- 问题：没有对照组，无法判断变化是否由AI引起
- 建议：设置对照组（不使用AI的学生）

⚠️ 样本量未说明
- 问题：需要多少样本？如何抽样？
- 建议：根据统计功效分析确定样本量，至少60-80人

改进方案：
采用准实验设计，招募100名学生，随机分为实验组（使用AI）和对照组（不使用AI）。在8周干预前后，使用批判性思维测试和写作样本评分来测量批判性思维能力的变化。
```

**参数说明**：
- **研究方法描述**（必填）：描述你的研究方法设计
- **研究问题**（必填）：你的研究问题，用于判断方法是否匹配

### 功能3：推荐分析方法

**用途**：根据数据类型和研究问题，推荐合适的数据分析方法。

**使用场景**：
- 已经收集了数据，不知道用什么方法分析
- 想要确保分析方法的科学性
- 需要学习新的分析方法

**使用示例**：
```
数据类型：定量数据

研究问题：
实验组和对照组的批判性思维能力是否有显著差异？

样本量：实验组60人，对照组60人

推荐分析方法：

主要方法：独立样本t检验
- 用途：比较两组独立样本的均值差异
- 前提条件：数据正态分布、方差齐性
- 软件：SPSS、R、Python

补充方法：
1. 描述性统计：计算均值、标准差、中位数
2. 正态性检验：Shapiro-Wilk检验
3. 方差齐性检验：Levene检验
4. 效应量计算：Cohen's d
5. 如果不满足参数检验条件，使用Mann-Whitney U检验（非参数检验）

报告示例：
"独立样本t检验结果显示，实验组的批判性思维得分（M=75.3, SD=8.2）显著高于对照组（M=68.5, SD=9.1），t(118)=4.32, p<0.001, Cohen's d=0.79，表明效应量较大。"
```

**参数说明**：
- **数据类型**（必填）：定量、定性、混合等
- **研究问题**（必填）：你想回答什么问题
- **样本量**（可选）：样本量会影响方法选择

## 使用技巧

1. **先问题后方法**：不要为了用某种方法而用，要根据问题选择方法
2. **考虑可行性**：理想的方法不一定可行，要考虑时间、资源、能力等限制
3. **学习经典研究**：找几篇本领域的经典研究，学习其方法设计
4. **咨询专家**：如果不确定，及时咨询导师或统计专家
5. **预实验**：正式研究前，可以做小规模预实验，测试方法的可行性

## 常见问题

**Q: 定量研究和定性研究有什么区别？**

A: 定量研究用数字和统计方法，关注"多少"、"是否有差异"；定性研究用文字和主题分析，关注"为什么"、"如何"。两者可以结合使用（混合研究）。

**Q: 什么时候用实验设计，什么时候用调查研究？**

A: 如果研究问题关注因果关系（X对Y的影响），应该用实验设计；如果关注现状、态度、关系，可以用调查研究。

**Q: 样本量多少合适？**

A: 取决于研究设计和统计方法。一般来说，实验研究每组至少30人，调查研究至少100人。可以用统计功效分析（power analysis）计算所需样本量。

**Q: 如何保证研究的信度和效度？**

A: 信度（可靠性）：使用标准化工具、多次测量、评分者一致性。效度（有效性）：确保测量的是你想测量的东西、控制混淆变量、使用多种测量方法。
