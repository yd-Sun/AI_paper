# 论文主题确定器

> 通过苏格拉底式追问，帮助用户从模糊想法收敛到明确的论文研究主题。持续发问直到主题清晰、可研究、有边界。

## 功能概述

很多研究者在论文写作初期都会遇到一个问题：有一个大致的想法，但不知道如何把它变成一个明确的研究主题。本技能采用**苏格拉底式追问法**，通过一系列有针对性的问题，帮助你：
- 从宽泛的兴趣领域聚焦到具体的研究主题
- 明确研究的边界和范围
- 确保主题具有可研究性和学术价值
- 避免主题过大或过小的问题

适合在论文写作最初期使用，特别是当你只有一个模糊的想法，不知道如何开始时。

## 主要功能

### 功能1：确定论文主题

**用途**：通过连续的追问和引导，帮助你把一个模糊的想法逐步细化为一个清晰、可研究的论文主题。

**使用场景**：
- 刚开始准备写论文，只有一个大致的兴趣方向
- 主题太宽泛，不知道如何缩小范围
- 主题太具体，担心没有足够的研究空间
- 需要确认主题是否具有研究价值

**使用示例**：
```
初步想法：我想研究AI对教育的影响

系统追问过程：
Q1: 你对AI在教育中的哪个具体应用更感兴趣？（如智能辅导、自动评分、个性化学习等）
A1: 我对AI辅助写作比较感兴趣

Q2: 你想研究哪个教育阶段的写作？（小学、中学、大学、研究生）
A2: 大学生的学术写作

Q3: 你想关注AI辅助写作的哪个方面？（效率提升、质量改善、能力培养、伦理问题等）
A3: 我想看看AI会不会影响学生的写作能力

Q4: 你说的"写作能力"具体指什么？（批判性思维、论证能力、语言表达、创新性等）
A4: 主要是批判性思维和论证能力

Q5: 你打算如何研究这个问题？（实验研究、问卷调查、案例分析等）
A5: 可能做实验，对比使用和不使用AI的学生

最终主题：
生成式AI工具对大学生学术写作中批判性思维能力的影响：一项准实验研究
```

**参数说明**：
- **你的初步想法**（必填）：用一两句话描述你感兴趣的研究方向，不需要很具体
- **学科领域**（可选）：选择你的学科，系统会根据学科特点调整追问方式
- **论文类型**（可选）：期刊论文、学位论文、课程论文等，不同类型对主题的要求不同

**注意事项**：
- 这是一个**互动过程**，需要多轮对话才能确定主题
- 回答问题时尽量具体，避免模糊的表述
- 如果某个问题不确定，可以说"不确定"，系统会提供选项
- 最终主题应该满足：清晰、可研究、有边界、有价值

## 使用技巧

1. **诚实表达你的兴趣**：不要试图迎合"热门话题"，选择你真正感兴趣的方向
2. **准备好多轮对话**：通常需要5-10轮追问才能确定一个好的主题
3. **记录追问过程**：这些问题和答案可以作为你研究背景的一部分
4. **不要急于确定**：如果对某个方向不确定，可以多尝试几次
5. **结合文献检索**：确定主题后，立即进行文献检索，验证主题的可行性

## 常见问题

**Q: 我的想法太宽泛了，怎么办？**

A: 这正是本技能要解决的问题！通过追问，系统会帮你逐步缩小范围。例如"AI对教育的影响"可以缩小到"AI对大学生学术写作批判性思维的影响"。

**Q: 我的想法太具体了，会不会没有研究空间？**

A: 系统会评估你的主题是否过于具体。如果是，会建议你适当扩大范围，或者调整研究角度。

**Q: 如果我对追问的问题不知道怎么回答怎么办？**

A: 可以回答"不确定"或"不知道"，系统会提供一些常见选项供你参考，或者换一个角度继续追问。

**Q: 确定主题后还能修改吗？**

A: 当然可以！主题确定不是一次性的，在文献阅读和研究过程中，主题可能会进一步调整。本技能可以多次使用。

**Q: 这个技能和"研究问题提炼器"有什么区别？**

A: 本技能帮你确定**研究主题**（一个大的方向），而"研究问题提炼器"帮你把主题转化为**具体的研究问题**（可以直接回答的问题）。建议先用本技能确定主题，再用研究问题提炼器。

## 使用技巧进阶

### 如何判断主题是否合适？

一个好的研究主题应该满足以下标准：

1. **清晰性**：别人能够理解你要研究什么
2. **可研究性**：有可行的研究方法可以使用
3. **有边界**：范围明确，不会无限扩大
4. **有价值**：对学术界或实践有贡献
5. **有兴趣**：你自己对这个主题感兴趣

### 不同论文类型的主题特点

- **课程论文**：主题可以相对简单，重点在于展示你对课程内容的理解
- **期刊论文**：需要有创新性，填补研究空白
- **硕士论文**：需要有一定深度，但不要求重大创新
- **博士论文**：需要有重大创新，对领域有显著贡献
