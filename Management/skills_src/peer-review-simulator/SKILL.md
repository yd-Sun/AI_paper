# 同行评审模拟器

> 模拟国际期刊的同行评审过程，从主编、方法论、领域专家、跨学科、魔鬼代言人五个视角进行评审，输出结构化的审稿意见和修改路线图。

## 功能概述

在投稿前，通过模拟同行评审可以帮你：
- 发现论文中的潜在问题
- 从多个角度审视研究质量
- 提前准备审稿人可能提出的问题
- 提高论文的接受率

本技能模拟真实的同行评审过程，从5个不同视角（主编、方法论专家、领域专家、跨学科审稿人、魔鬼代言人）对论文进行全面评审。

## 主要功能

### 功能1：模拟同行评审

**用途**：从5个不同视角模拟完整的同行评审过程，提供详细的审稿意见和修改建议。

**使用场景**：
- 论文完成，准备投稿前
- 想要提前发现论文中的问题
- 需要从多个角度审视研究质量

**使用示例**：
```
论文内容：
[粘贴论文的摘要、引言、方法、结果、讨论等核心部分]

论文类型：实证研究

目标期刊：Educational Technology Research and Development

审稿意见：

【主编视角】整体评估
总体评价：Minor Revision（小修）
- 研究主题符合期刊范围 ✓
- 研究设计基本合理 ✓
- 写作质量良好 ✓
- 创新性有待加强 ⚠️

主要问题：
1. 研究的创新点不够突出，需要在引言中明确说明
2. 讨论部分与已有研究的对比不够充分
3. 研究局限性部分过于简略

【方法论专家视角】
评分：7/10
优点：
- 研究设计合理，采用准实验设计
- 样本量充足（n=120）
- 数据分析方法恰当

问题：
1. ❌ 缺少样本量计算的依据（统计功效分析）
2. ⚠️ 控制变量的选择需要更充分的理论支持
3. ❌ 测量工具的信效度报告不完整
4. ⚠️ 数据收集程序描述不够详细

建议：
- 补充样本量计算过程
- 报告测量工具的Cronbach's α系数
- 详细描述数据收集的具体步骤和时间安排

【领域专家视角】
评分：8/10
优点：
- 文献综述全面，涵盖了主要研究
- 研究问题有实际意义
- 结果分析深入

问题：
1. ⚠️ 遗漏了几篇重要的近期研究（2023-2024）
2. ❌ 对"批判性思维"的操作化定义不够清晰
3. ⚠️ 讨论部分对实践意义的阐述不够具体

建议：
- 补充最新的相关研究
- 明确批判性思维的具体维度和测量方式
- 增加对教育实践的具体建议

【跨学科审稿人视角】
评分：6/10
优点：
- 研究问题清晰
- 结构完整

问题：
1. ❌ 对非教育学背景的读者不够友好，专业术语过多
2. ⚠️ 缺少对AI技术本身的介绍
3. ❌ 研究的理论框架不够明确

建议：
- 在引言中增加对生成式AI的基本介绍
- 明确研究的理论框架（如认知负荷理论、建构主义等）
- 适当解释专业术语

【魔鬼代言人视角】挑战性问题
1. 你如何确保实验组和对照组的基线一致？如果本来就有差异呢？
2. 8周的干预时间是否足够？长期效果如何？
3. 学生可能在实验外也使用AI，如何控制？
4. 批判性思维的测量是否真的反映了实际能力？
5. 研究结果是否可以推广到其他学校、其他学科？

【综合评审意见】
推荐决定：Minor Revision（小修后接受）

必须修改的问题（Major Issues）：
1. 补充测量工具的信效度报告
2. 明确批判性思维的操作化定义
3. 补充样本量计算依据

建议修改的问题（Minor Issues）：
1. 补充最新文献
2. 增加对AI技术的介绍
3. 完善研究局限性讨论
4. 增加实践建议的具体性

修改路线图：
第一步（1-2天）：补充方法部分的技术细节
第二步（2-3天）：完善文献综述，补充最新研究
第三步（1-2天）：优化讨论部分，增加实践意义
第四步（1天）：回应魔鬼代言人的挑战性问题
```

**参数说明**：
- **论文内容**（必填）：粘贴论文的核心部分
- **论文类型**（可选）：实证研究、综述、理论研究等
- **目标期刊**（可选）：如果有目标期刊，评审会更有针对性

**注意事项**：
- 模拟评审不能替代真实的同行评审
- 建议在论文基本完成后使用，而不是初稿阶段
- 评审意见仅供参考，最终以真实审稿意见为准

### 功能2：快速质量评估

**使用场景**：
- 想要快速了解论文质量
- 时间紧迫，需要快速反馈
- 初稿完成，需要初步评估

**使用示例**：
```

[粘贴摘要和核心部分]

快速评估结果：

【研究问题】★★★★☆
- 清晰度：问题表述清晰
- 重要性：有实际意义
- 新颖性：有一定创新，但不够突出

【研究方法】★★★☆☆
- 设计合理性：基本合理
- 可行性：可行
- 严谨性：部分细节需要完善

【结果呈现】★★★★☆
- 清晰度：结果清晰
- 完整性：基本完整
- 可信度：数据可信

【讨论质量】★★★☆☆
- 深度：需要加强
- 与文献的对比：不够充分
- 实践意义：需要更具体

【写作质量】★★★★☆
- 逻辑性：逻辑清晰
- 语言表达：流畅
- 结构完整性：完整

总体评分：7.2/10
建议：重点改进研究方法的细节和讨论的深度
```

**参数说明**：
- **论文内容**（必填）：至少包含摘要和核心部分

### 功能3：方法论审查

**用途**：专注于研究设计、数据分析和可重复性的审查。

**使用场景**：
- 对方法部分不确定
- 审稿人对方法提出质疑
- 想要确保方法的严谨性

**使用示例**：
```
方法论部分：
[粘贴方法部分]

结果部分：
[粘贴结果部分]

方法论审查：

【研究设计】
✓ 设计类型明确（准实验设计）
✓ 有对照组
⚠️ 随机化程序描述不够详细
❌ 缺少盲法说明

【样本】
✓ 样本量充足
⚠️ 抽样方法需要说明
❌ 缺少样本代表性分析

【测量工具】
✓ 工具选择合理
❌ 缺少信效度报告
❌ 缺少评分者信度

【数据分析】
✓ 分析方法恰当
✓ 统计检验正确
⚠️ 效应量未报告
⚠️ 缺少多重比较校正

【可重复性】
❌ 数据收集程序不够详细
❌ 缺少原始数据或补充材料
⚠️ 分析代码未提供

改进建议：
1. 详细描述随机化和盲法程序
2. 报告所有测量工具的信效度
3. 补充效应量（Cohen's d或η²）
4. 考虑提供补充材料或数据
```

**参数说明**：
- **方法论部分**（必填）：论文的方法部分
- **结果部分**（可选）：有助于评估方法与结果的一致性

## 使用技巧

1. **在投稿前使用**：给自己留出修改时间
2. **认真对待每条意见**：即使是模拟评审，也要认真对待
3. **准备回应**：对每条意见准备回应，这在真实评审中也会用到
4. **多次使用**：修改后可以再次使用，确保问题已解决
5. **结合真实评审**：如果收到真实审稿意见，可以对比参考

## 常见问题

**Q: 模拟评审和真实评审有什么区别？**

A: 模拟评审基于通用的学术标准，真实评审会考虑期刊的具体要求和审稿人的个人偏好。模拟评审可以帮你发现大部分问题，但不能完全替代真实评审。

**Q: 如果模拟评审结果不好，是不是不应该投稿？**

A: 不一定。模拟评审的目的是帮你改进论文，而不是劝退。如果发现问题，修改后再投稿即可。

**Q: 五个视角都要看吗？**

A: 建议都看。不同视角关注不同方面，全面审视可以发现更多问题。

**Q: 魔鬼代言人的问题太刁钻了，怎么办？**

A: 这些问题虽然刁钻，但都是真实审稿人可能提出的。提前准备好回应，可以提高论文质量和投稿成功率。
