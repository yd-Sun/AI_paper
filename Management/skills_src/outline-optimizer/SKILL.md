# 大纲优化助手

> 强化论文大纲生成质量，补充研究问题、方法路径与章节层级约束。

## 功能概述

论文大纲是整篇论文的骨架，一个好的大纲能让写作事半功倍。本技能帮助你：
- 检查大纲的结构完整性
- 确保各章节之间的逻辑关系
- 补充缺失的重要部分
- 调整章节层级和顺序
- 确保大纲与研究问题的一致性

适合在完成初步大纲后使用，通过专业的审校，让你的大纲更加科学、完整。

## 主要功能

### 功能1：审校大纲

**用途**：根据研究主题和现有大纲，给出结构改进建议，包括章节调整、内容补充、逻辑优化等。

**使用场景**：
- 刚完成大纲初稿，需要专业审校
- 导师对大纲提出修改意见，需要优化
- 感觉大纲不够完善，但不知道问题在哪

**使用示例**：
```
研究主题：
生成式AI工具对大学生学术写作中批判性思维能力的影响

现有大纲：
1. 引言
2. 文献综述
3. 研究方法
4. 研究结果
5. 讨论
6. 结论

审校建议：

结构问题：
❌ 引言部分过于简单，建议细化为：
  1.1 研究背景
  1.2 研究问题
  1.3 研究意义
  1.4 论文结构

❌ 文献综述缺少逻辑框架，建议调整为：
  2.1 生成式AI在教育中的应用
  2.2 批判性思维的定义与测量
  2.3 AI对学习能力影响的相关研究
  2.4 研究空白与本研究的定位

✅ 研究方法部分结构合理，建议补充：
  3.1 研究设计（准实验设计）
  3.2 研究对象与样本
  3.3 研究工具与材料
  3.4 数据收集程序
  3.5 数据分析方法

⚠️ 结果与讨论建议分开，避免混淆：
  4. 研究结果
    4.1 描述性统计
    4.2 假设检验结果
  5. 讨论
    5.1 主要发现的解释
    5.2 与已有研究的对比
    5.3 研究的理论与实践意义
    5.4 研究局限性

优化后的大纲：
1. 引言
  1.1 研究背景
  1.2 研究问题
  1.3 研究意义
  1.4 论文结构
2. 文献综述
  2.1 生成式AI在教育中的应用
  2.2 批判性思维的定义与测量
  2.3 AI对学习能力影响的相关研究
  2.4 研究空白与本研究的定位
3. 研究方法
  3.1 研究设计
  3.2 研究对象与样本
  3.3 研究工具与材料
  3.4 数据收集程序
  3.5 数据分析方法
4. 研究结果
  4.1 描述性统计
  4.2 假设检验结果
5. 讨论
  5.1 主要发现的解释
  5.2 与已有研究的对比
  5.3 理论与实践意义
  5.4 研究局限性
6. 结论与建议
  6.1 主要结论
  6.2 研究贡献
  6.3 未来研究方向
参考文献
附录
```

**参数说明**：
- **研究主题**（必填）：你的研究主题，用于判断大纲是否与主题一致
- **现有大纲**（必填）：粘贴你当前的论文大纲

**注意事项**：
- 大纲应该包含到二级或三级标题
- 不同学科的大纲结构可能有差异，建议参考本学科的经典论文
- 大纲不是一成不变的，在写作过程中可以调整

## 使用技巧

1. **先粗后细**：先确定一级标题，再细化二级、三级标题
2. **保持逻辑性**：各章节之间应该有清晰的逻辑关系，前后呼应
3. **参考经典论文**：找几篇本领域的优秀论文，学习其大纲结构
4. **与导师沟通**：大纲确定后，及时与导师沟通，获得反馈
5. **定期更新**：在写作过程中，根据实际情况调整大纲

## 常见问题

**Q: 论文大纲一般包含哪些部分？**

A: 典型的学术论文大纲包括：引言、文献综述、研究方法、研究结果、讨论、结论、参考文献。不同学科可能有差异，如理工科可能有"理论分析"部分，人文社科可能有"案例分析"部分。

**Q: 大纲需要细化到什么程度？**

A: 建议至少细化到三级标题。一级标题是章，二级标题是节，三级标题是具体内容点。过于粗略的大纲指导性不强，过于详细的大纲又失去了灵活性。

**Q: 文献综述应该怎么组织？**

A: 文献综述不是简单罗列文献，而是要有逻辑框架。常见的组织方式有：1) 按主题分类；2) 按时间顺序；3) 按理论流派；4) 按研究方法。最后要指出研究空白，引出你的研究。

**Q: 结果和讨论可以合并吗？**

A: 不建议。结果部分应该客观呈现数据和发现，讨论部分才是解释和分析。合并容易导致逻辑混乱。但在某些学科（如定性研究）中，结果和讨论可以交织进行。

**Q: 大纲确定后还能修改吗？**

A: 可以，而且很正常。在写作过程中，你可能会发现某些部分需要调整。但不建议频繁大改，否则会影响写作进度。

## 不同论文类型的大纲特点

### 实证研究论文
- 强调研究方法和结果
- 需要详细描述研究设计、样本、工具、程序
- 结果部分要有数据支撑

### 综述论文
- 文献综述是核心部分，占比最大
- 需要系统梳理研究现状
- 最后提出未来研究方向

### 理论研究论文
- 强调理论分析和逻辑推理
- 可能没有实证数据
- 需要有清晰的理论框架

### 案例研究论文
- 需要详细描述案例背景
- 案例分析是核心部分
- 要有理论与实践的结合
