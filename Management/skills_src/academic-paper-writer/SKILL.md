# AI 论文助手

> 12 个 AI 代理协作的学术论文写作系统，支持从选题到成稿的全流程写作。

## 功能概述

AI 论文助手是一个完整的学术论文写作工具，由 12 个专业 AI 代理组成：

| # | 代理 | 角色 | 职责 |
|---|------|------|------|
| 1 | 配置顾问 | intake_agent | 收集论文参数，输出配置记录 |
| 2 | 文献策略师 | literature_strategist | 设计检索策略，筛选文献来源 |
| 3 | 结构建筑师 | structure_architect | 设计论文结构，生成大纲 |
| 4 | 论证构建师 | argument_builder | 构建论证逻辑，处理反驳 |
| 5 | 写作专家 | draft_writer | 撰写全文，调整语域 |
| 6 | 引用合规官 | citation_compliance | 检查引用格式，验证完整性 |
| 7 | 摘要专家 | abstract_bilingual | 生成双语摘要 |
| 8 | 同行评审员 | peer_reviewer | 模拟双盲评审 |
| 9 | 格式化师 | formatter | 格式转换，期刊模板 |
| 10 | 苏格拉底导师 | socratic_mentor | 引导式写作规划 |
| 11 | 可视化专家 | visualization | 生成学术图表 |
| 12 | 修订教练 | revision_coach | 指导论文修改 |

## 支持的模式

| 模式 | 触发方式 | 功能 |
|------|----------|------|
| `full` | "写论文" | 完整论文写作流程 |
| `plan` | "规划论文" | 苏格拉底式引导规划 |
| `outline-only` | "生成大纲" | 仅生成论文大纲 |
| `lit-review` | "文献综述" | 系统化文献检索 |
| `abstract-only` | "生成摘要" | 生成双语摘要 |
| `revision` | "修订论文" | 根据评审意见修改 |
| `revision-coach` | "修订指导" | 解析评审意见 |
| `citation-check` | "检查引用" | 验证引用格式 |
| `format-convert` | "格式转换" | 转换输出格式 |
| `disclosure` | "AI 声明" | 生成 AI 使用声明 |

## 支持的论文类型

- 实证研究论文 (IMRaD)
- 文献综述论文
- 案例研究论文
- 理论研究论文
- 会议论文
- 政策简报

## 支持的引用格式

- APA 第 7 版
- GB/T 7714 (国标)
- 芝加哥格式
- MLA 第 9 版
- IEEE
- 温哥华格式

## 支持的输出格式

- Markdown
- LaTeX (.tex + .bib)
- DOCX (via Pandoc)
- PDF

## 使用示例

### 示例 1：完整论文写作

```
我想写一篇关于 AI 对高等教育质量保障影响的论文
```

AI 会：
1. 确认论文配置（类型、学科、引用格式等）
2. 设计文献检索策略
3. 生成论文大纲
4. 构建论证逻辑
5. 撰写全文
6. 检查引用格式
7. 生成双语摘要
8. 进行同行评审
9. 格式化输出

### 示例 2：论文规划

```
帮我规划一篇关于出生率下降对民办高校管理策略影响的论文
```

AI 会通过苏格拉底式提问引导你：
1. 明确研究问题
2. 评估研究准备度
3. 设计论文结构
4. 提取核心洞见

### 示例 3：文献综述

```
帮我检索关于数字化转型对传统制造业影响的文献
```

AI 会：
1. 设计检索策略（关键词、数据库）
2. 筛选文献来源
3. 生成带注释的文献列表
4. 构建文献矩阵

## 质量标准

### 写作质量
- 每个主张必须有引用或数据支撑
- 避免 AI 典型用词
- 段落长度多样化（2-8句）
- 使用学科专业术语

### 引用质量
- 100% 格式合规
- 所有 DOI 必须包含
- 标记超过10年的文献
- 自引比例不超过15%

### 评审标准
- 原创性 (20%)
- 方法论严谨性 (25%)
- 证据充分性 (25%)
- 论证连贯性 (15%)
- 写作质量 (15%)
