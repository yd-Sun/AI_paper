# 图表模板库

> 提供常用学术图表模板：实验流程、研究框架、系统架构、ER 模型、用户旅程等。在编辑器中可一键插入并由 AI 填充内容。

## 功能概述

从零开始绘制图表往往比较困难。本技能提供了一系列常用的学术图表模板，你可以：
- 浏览各种类型的图表模板
- 一键插入模板到编辑器
- 让AI根据你的需求填充内容
- 快速生成专业的学术图表

适合在需要绘制图表但不知道如何开始时使用。

## 支持的图表类型

本技能提供以下类型的图表模板：
- **流程图（Flowchart）**：研究流程、实验流程、数据处理流程等
- **时序图（Sequence）**：系统交互、用户操作流程等
- **ER图（ER Diagram）**：数据库设计、实体关系等
- **类图（Class Diagram）**：系统架构、模块关系等
- **状态图（State Diagram）**：状态转换、生命周期等
- **思维导图（Mindmap）**：概念框架、知识结构等

## 主要功能

### 功能1：列出模板

**用途**：按图表类型列出当前可用的模板。

**使用场景**：
- 想要了解有哪些可用的模板
- 寻找适合自己研究的图表类型
- 浏览模板获取灵感

**使用示例**：
```
图表类型：流程图

可用模板：

【模板1：研究流程图】
适用场景：展示完整的研究过程
包含元素：
- 文献综述
- 研究设计
- 数据收集
- 数据分析
- 结果讨论
- 结论

【模板2：实验流程图】
适用场景：展示实验的具体步骤
包含元素：
- 实验准备
- 前测
- 干预/处理
- 后测
- 数据分析

【模板3：数据处理流程图】
适用场景：展示数据处理的步骤
包含元素：
- 原始数据
- 数据清洗
- 数据转换
- 数据分析
- 结果输出

【模板4：问卷调查流程图】
适用场景：展示问卷调查的完整流程
包含元素：
- 问卷设计
- 预测试
- 正式发放
- 数据收集
- 数据分析
```

**参数说明**：
- **图表类型**（可选）：选择特定类型，或选择"全部"查看所有模板

## 使用方法

### 步骤1：浏览模板
使用"列出模板"功能，浏览可用的图表模板，找到适合你研究的模板。

### 步骤2：选择模板
在图表生成场景中，告诉AI你想使用哪个模板，例如：
"使用研究流程图模板，生成我的研究流程图"

### 步骤3：提供信息
提供你的具体研究信息，让AI填充模板内容，例如：
"我的研究是关于AI对学生写作能力的影响，采用准实验设计..."

### 步骤4：调整优化
生成后，可以要求AI调整细节，如：
"把数据分析部分展开，增加具体的分析方法"

## 使用技巧

1. **先选模板，再定制**：不要从零开始，先找一个接近的模板
2. **提供详细信息**：告诉AI你的具体情况，让AI更好地填充内容
3. **迭代优化**：生成后可以多次调整，直到满意
4. **保存常用模板**：如果某个模板经常用，可以保存下来
5. **参考文献**：看看同领域的论文用了什么图表

## 常见问题

**Q: 如何选择合适的图表类型？**

A: 根据你要表达的内容：
- 展示流程/步骤 → 流程图
- 展示交互/顺序 → 时序图
- 展示关系/结构 → ER图或类图
- 展示状态变化 → 状态图
- 展示概念层级 → 思维导图

**Q: 模板可以修改吗？**

A: 可以。模板只是起点，你可以根据需要自由修改。

**Q: 如果没有合适的模板怎么办？**

A: 可以选择一个接近的模板，然后大幅修改；或者直接让AI从零开始生成。

**Q: 模板会定期更新吗？**

A: 会的。我们会根据用户反馈和常见需求，定期添加新的模板。
